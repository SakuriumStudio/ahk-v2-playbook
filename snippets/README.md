# Snippets 入口（コピペで動く最小単位）

ここは「そのまま動く」小さなスクリプト置き場。  
**目的→該当スニペット**の順で探す。

---

## まず使うやつ

- `check_version.ahk`：v2で動いてるか確認
- `debug_key_viewer.ahk`：押してるキー/入力が取れてるか確認

---

## UI / 常駐

- `always_on_top_overlay.ahk`：小窓を最前面に固定（ドラッグ移動）

---

## 調査・運用

- `process_exe_finder.ahk`：今アクティブなアプリの exe 名を表示（Synapse登録の時に便利）

---

## 追加すると便利なカテゴリ（今後増やす用）

- window操作（取得/移動/サイズ変更）
- クリック透過（クリックは下へ、表示だけ上）
- ホットキーの衝突回避（#HotIf / 条件分岐）
- ログ（FileAppend / OutputDebug）
